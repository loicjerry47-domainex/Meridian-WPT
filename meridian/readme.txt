A conversion-focused WordPress theme for SaaS, agencies, and startups.

== Description ==
Meridian prioritizes landing page design, lead generation, and showcasing products/services. Includes a Dark Mode toggle, custom post types for Testimonials, and pricing tables via shortcode.

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and choose the file meridian.zip.
3. Click Install Now.
4. Click Activate to use your new theme right away.

== Frequently Asked Questions ==
= Does it support dark mode? =
Yes! Meridian supports a fully synced dark mode toggle stored in localStorage.
